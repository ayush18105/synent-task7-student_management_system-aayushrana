Student Management System

Description

A Python-based Student Management System that allows users to manage student records efficiently.

Features

* Add Student
* Update Student
* Delete Student
* View All Students
* Search Student
* Data Storage using CSV/JSON

Technologies Used

* Python 3
* CSV / JSON File Handling

Project Structure

```
Student_Management_System/
│
├── main.py
├── students.csv
├── students.json
├── README.md
```

How to Run

1. Clone the repository:

```bash
git clone https://github.com/ayush18105/synent-task7-student_management_system-aayushrana.git
```

2. Navigate to the project folder:

```bash
cd synent-task7-student_management_system-aayushrana
```

3. Run the program:

```bash
python main.py
```
